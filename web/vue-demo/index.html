业务逻辑说明

活跃告警

来自 Alertmanager 仍处于 firing 状态的告警。告警持续触发时会定期推送，直到恢复或手动屏蔽。

活跃事件

将同一规则、同一级别的多条告警聚合成一个"事件"，按 severity 降序，与推送到飞书/钉钉/企微的消息内容一致。

屏蔽规则

在生效时间段内，匹配告警不再推送通知。可指定告警名称、主机、实例或原因说明。

冷却抑制

同一告警在 cooldown 周期内重复触发时，仅记录历史但不重复推送，避免通知轰炸。

通知状态

已通知：成功推送到通道；未通知：firing 但未推送；已屏蔽：命中屏蔽规则；冷却抑制：在冷却周期内。

修复状态

已修复：Alertmanager 已发送 resolved；未修复：告警仍在 firing 或未收到恢复通知。
